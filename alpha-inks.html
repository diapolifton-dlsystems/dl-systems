Alpha Inks Tattoo Studio

Production website for a real tattoo studio — designed, engineered, and deployed for real‑world use.

Project Overview

Alpha Inks is a real tattoo studio requiring a professional web presence to showcase work, communicate services, and allow clients to initiate contact easily across devices.

The objective was not experimental design, but a clean, fast, and reliable production website that reflects the studio’s brand while remaining easy to maintain.

Scope of Work

  • Frontend website architecture and UI implementation
  • Responsive design for mobile, tablet, and desktop
  • SEO‑ready structure and metadata
  • Contact and enquiry functionality
  • Deployment and live hosting

Technology Stack

  • HTML5 — semantic structure
  • CSS3 — layout, responsiveness, visual styling
  • JavaScript — client‑side interactivity
  • Firebase — backend services
  • EmailJS — enquiry handling

Engineering Considerations

The site was engineered with clarity and performance in mind. No unnecessary frameworks were introduced, ensuring fast load times and predictable behaviour.

The system design prioritises:

  • Low operational complexity
  • Ease of updates and content changes
  • Consistent behaviour across browsers
  • Direct communication between client and studio
